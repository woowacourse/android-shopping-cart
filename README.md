# android-shopping-cart

## 1단계 기능 요구 사항

- 상품 목록
    - [x] 상품에 이름, 가격, 이미지 url이 있다.
    - [x] 상품을 클릭하면 상품 상세로 이동한다.
    - [x] 앱바 장바구니 이모티콘을 누르면, 장바구니로 이동한다.
- 상품 상세
    - [x] 상품 이미지, 이름, 가격, 버튼을 표시한다.
    - [x] 상품 상세에서 장바구니에 상품을 담을 수 있다.
    - [x] 앱바에 X를 클릭하면, 상품 목록 화면으로 이동한다.
- 장바구니
    - [x] 상품 이름, 가격, 이미지, 버튼을 표시한다.
    - [x] 장바구니에서 원하는 상품을 삭제할 수 있다.
    - [x] 앱바의 뒤로 가기를 클릭하면, 상품 목록 화면으로 이동한다.

## 2단계 기능 요구 사항

- 데이터 로딩 전략을 적용한다.
    - [x] 상품 목록에서 더보기 버튼을 눌러 추가 로드할 수 있다. (10개 기준)
    - [x] 장바구니 목록은 페이지네이션이 되어야 한다. (5개 기준)

## 3단계 기능 요구 사항

### 상품 목록 페이지

1. 앱바에서 장바구니에 담은 개수를 표시

- [X] 장바구니에서 제품이 들어간 개수를 바탕화면에서 확인할 수 있다.
- [ ] 장바구니에서 100개가 넘으면 99라고 표시된다.

2. 상품 목록 - 수량 조절

- [ ] 제품 화면에 플러스 버튼을 누르면 그 버튼은 사라진다
- [ ] 플러스 버튼을 누르면, 수량 입력 버튼으로 바뀐다.
- [ ] 플러스 버튼을 누르면, 장바구니에 상품이 추가된다
- [ ] 수량 입력 버튼을 +, - 할 수 있다.
- [ ] -는 1미만으로 내려가지 않는다.
- [ ] 숫자가 1인 상태에서 -를 누르면, 장바구니에서 상품이 삭제된다.

3. 상품 상세 - 수량 조절

- [ ] 상품 목록에 있는 수량 조절 버튼을 재활용한다.
- [ ] 수량을 +, -하면, 가격이 계산된다.
- [ ] 장바구니 담기를 누르면, 장바구니에 상품이 추가된다.
- [ ] 상품 목록 페이지에서 수량을 조절했고 그것이 장바구니에 들어간 상태이면, 그 상태를 반영하여, 상품 상세에 수량을 표시한다.

4. 장바구니 - 수량 조절

- [ ] 상품 목록에 있는 수량 조절 버튼을 재활용한다.
- [ ] 수량을 +, -하면, 가격이 계산된다.
- [ ] -는 1미만으로 내려가지 않고 비활성화된다
- [ ] 수량을 +, -하면, 가격이 계산된다.
