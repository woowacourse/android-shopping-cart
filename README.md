# android-shopping-cart

# 🚀 1단계 - 상품 목록 & 2단계 - 데이터 로딩

# 도메인

## 물건

- 키(id)
- 사진
- 이름
- 가격

## 물건 리스트

- 물건 여러개

## 장바구니 물건

- 키(id)
- 물건 키(id)
- (나중에) 장바구니 물건의 개수
- (나중에) 장바구니 물건에 대한 가격

## 장바구니

- 장바구니 물건 리스트
- 동일한 물건을 장바구니에 넣었을 때 중복으로 들어간다.
- (나중에) 동일한 물건을 장바구니에 넣었을 때 동일한 장바구니 물건의 개수가 늘어난다.
- 물건 리스트를 가지고 있어야 한다. (물건이 없으면 빈 리스트) → 담겨져있는 상품이 없습니다.
- 장바구니에서 물건을 삭제할 수 있다.
- 장바구니에 물건을 추가할 수 있다.

# UI

## 물건 목록 페이지

- 상단바에 장바구니 아이콘과 Shopping 텍스트가 있다.
- 물건 이름의 길이는 한 줄을 초과하면 안된다.
    - 물건 이름의 길이가 한 줄을 초과한 경우 … 으로 표기한다.
- 가격을 나타낼 때 세자리마다 콤마를 찍는다.
- 물건 리스트가 보여야한다.
- 물건을 클릭했을 때 물건의 상세정보 페이지가 보여야 한다.
- 장바구니 아이콘 버튼을 눌렀을 때 장바구니 물건 리스트 페이지가 보여야한다.
- 물건이 20개만 나타나야 한다.
- 더보기 버튼을 눌렀을 때 물건 20개가 추가로 로드된다.

## 물건 상세 페이지

- 물건 상세 페이지에는 물건 사진, 이름, 가격, 장바구니 담기 버튼이 있어야한다.
- 물건 상세 페이지에서 상단바에 있는 x 버튼을 눌렀을 경우 물건 상세 페이지를 닫는다.
- 장바구니 담기 버튼을 누르면 장바구니 물건 리스트에 물건이 추가된다.

## 장바구니 페이지

- 상단바에는 뒤로가기 버튼과 Cart라는 텍스트가 존재한다.
- 장바구니 물건 리스트를 보여줄 때 한 줄에 2개만 나타나야 한다.
- 장바구니 물건 리스트 페이지에서 x버튼을 눌렀을 때 물건을 장바구니 물건 리스트에서 삭제된다.
- 장바구니 물건 리스트의 물건 개수가 5개 초과일 경우 5개를 기준으로 페이지를 나눈다.
- 화살표 버튼을 통해 이전 페이지로 돌아갈 수 있다.
- 장바구니 상세 페이지에서 첫 번째 페이지일 경우 왼쪽 화살표 버튼이 비활성화 된다.
- 장바구니 상세 페이지에서 마지막 페이지일 경우 오른쪽 화살표 버튼이 비활성화 된다.

# 도메인 테스트 시나리오

## 장바구니 테스트

- 동일한 물건을 장바구니에 넣었을 때 중복으로 들어간다.
- 물건1을 장바구니에 추가했을 때 물건1이 장바구니 물건 리스트에 추가된다.
- 장바구니에서 물건1을 삭제했을 때 장바구니 물건 리스트에서 물건1이 존재하지 않는다.

# UI 테스트 시나리오

## 물건 목록 페이지 테스트

- 장바구니 버튼을 눌렀을 때 장바구니 페이지가 열린다.
- 물건1을 눌렀을 때 물건1에 대한 상세 페이지가 열린다.
- 처음 더보기 버튼을 눌렀을 때 물건의 개수가 20개에서 40개가 되어야한다.

## 물건 상세 페이지 테스트

- 물건1의 상품 상세 페이지에서 장바구니 담기 버튼을 눌렀을 때, 장바구니 물건 리스트에 물건1이 추가된다.

## 장바구니 페이지

- 장바구니 페이지에 존재하는 물건1의 x 버튼을 눌렀을 경우 장바구니 안에는 물건1이 삭제된다.
- 장바구니의 페이지가 여러개이면서 현재 페이지가 첫 페이지인 경우 왼쪽 화살표 버튼이 비활성화 된다.
- 장바구니의 페이지가 여러개이면서 현재 페이지가 마지막 페이지인 경우 오른쪽 화살표 버튼이 비활성화 된다.
- 페이지가 한개인 경우 화살표가 존재하지 않는다.

# 파일 구조

## Product(data class)

- id: String
- imageUrl: ImageUrl
- name: ProductName
- price: Price

## ImageUrl

- value: String
- 빈 값이면 안된다.(notEmpty)

## ProductName

- value: String
- 빈 값이면 안된다.(notEmpty)

## Price

- value: Int
- 음수일 수 없다.

## Products

- value: List<Product>

## CartItem

- id: String
- product: Product

## CartItems

- value: List<CartItem>
- addCartItem()
- removeCartItem()

## Cart

- cartItems: CartItems
- addCartItem()
- removeCartItem()
