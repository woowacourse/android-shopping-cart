# Android Shopping Cart

## 구현 기능 목록

### 1. 상품 상세 (Product)

- [ ] 각 상품마다 고유한 UUID를 가진다
- [ ] 상품 이미지를 가진다
- [ ] 상품 이름을 가진다
- [ ] 상품 가격을 가진다

### 2. 장바구니 (Cart)

- [ ] 장바구니에 담긴 상품 목록을 관리(추가,삭제)한다
- [ ] 각 상품은 다음 정보를 포함한다
   - [ ] 상품 이미지
   - [ ] 상품 이름
   - [ ] 상품 가격

### 3.상품과 갯수 (ProductAndCount)

- [ ] 상품 이미지를 가진다
- [ ] 상품 이름을 가진다
- [ ] 상품 가격을 가진다
- [ ] 상품 수량을 조절할 수 있다.

### 4. 가격 (Price)

- [ ] 가격은 0보다 커야 한다
