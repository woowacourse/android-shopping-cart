# android-shopping-cart

## 1단계 구현 기능

## 도메인

### 상품(Product)

- [x]  id를 갖는다.
- [x]  이름을 갖는다.
- [x]  가격을 갖는다.
- [x]  이미지 URL을 갖는다. (String)

### 장바구니 아이템 (CartItem)

- [x]  상품(Product)를 갖는다.
- [x]  개수를 갖는다.
- [x]  총 가격을 안다.

### 장바구니(Cart)

- [ ]  장바구니 아이템 리스트를 갖는다. → UI 표시용
- [ ]  장바구니 아이템을 추가, 삭제 메시지를 보낼 수 있다.

### 상품 리스트 관리 객체(ProductManager)

- [ ]  상품(Product) 리스트를 갖는다.
- [ ]  아이디를 통해 상품을 전달한다.

## UI

## 상품 스크린(ProductScreen)

- [ ]  메인 탑바를 표시한다. (Shopping, CartIcon)
- [ ]  상품 카드들을 표시한다.

### 상품 카드(ProductCard)

- [ ]  상품 사진을 표시한다.
- [ ]  제목을 표시한다. (오버플로우)
- [ ]  가격 정보를 표시한다.

### 메인 탑바(MainTopBar)

- [ ]  제목, 우측 아이콘, 클릭 이벤트를 파라미터로 받는다.

## 상품 상세 스크린 (ProductDetailScreen)

- 닫기 버튼이 있는 탑바를 표시한다.
- 상품 이미지를 표시한다.
- 상품 정보 텍스트를 표시한다.
- 장바구니 담기 버튼을 표시한다.

### 장바구니 담기 버튼 (AddCartButton)

- [ ]  클릭 이벤트를 파라미터로 받는다.

### 상품 정보 텍스트 (ProductPrice)

- [ ]  상품 이름을 표시한다.
- [ ]  상품 가격 정보를 표시한다.

### 닫기 버튼이 있는 탑바 (DismissTopBar)

- [ ]  닫기 버튼 클릭 이벤트를 파라미터로 받는다.
- [ ]  닫기 버튼을 표시한다.

## 장바구니 스크린 (CartScreen)

- [ ]  카트 아이템 카드를 표시한다.
- [ ]  뒤로가기 버튼이 있는 탑바를 표시한다.

### 카트 아이템 카드 (CartItemCard)

- [ ]  상품 이름, 가격, 이미지를 표시한다.
- [ ]  닫기 버튼을 표시한다.
- [ ]  닫기 버튼 클릭 이벤트를 파라미터로 받는다.

### 뒤로가기 버튼이 있는 탑바 (NavigateUpTopBar)

- [ ]  뒤로가기 버튼을 표시한다.
- [ ]  타이틀을 표시한다.
- [ ]  뒤로가기 클릭 이벤트와 타이틀을 파라미터로 받는다.
