# Android Shopping Cart

## 1단계 구현 기능 목록

### 1. 상품 상세 (Product)

- [x] 각 상품마다 고유한 UUID를 가진다
- [x] 상품 이미지를 가진다
- [x] 상품 이름을 가진다
- [x] 상품 가격을 가진다

### 2. 장바구니 (Cart)

- [x] 장바구니에 담긴 상품 목록을 관리(추가,삭제)한다
- [x] 각 상품은 다음 정보를 포함한다
   - [x] 상품 이미지
   - [x] 상품 이름
   - [x] 상품 가격

### 3.상품과 갯수 (ProductAndCount)

- [x] 상품 이미지를 가진다
- [x] 상품 이름을 가진다
- [x] 상품 가격을 가진다
- [x] 상품 수량을 조절할 수 있다.

### 4. 가격 (Price)

- [x] 가격은 0보다 커야 한다

## 2단계 기능 구현 목록

### 1. Cart

- [x] 상품 목록에서 한 번에 20개씩 상품을 띄워주는데 더보기 버튼을 눌러 20개씩 추가 로드할 수 있도록 한다.
- [x] Cart에서 아이템이 5개 이하이면 아래에 패이지네이션이 안보이도록 한다.
- [x] 아이템이 5개 초과이면 아래에 페이지네이션이 보이도록 한다. 
- [x] 첫 페이지이면  왼쪽 버튼의 배경색이 회색(비활성화) 오른쪽은 민트색(활성화) 
- [x] 마지막 페이지이면 왼쪽 버튼의 배경색이 민트색(활성화) 오른쪽 버튼의 색이 회색(비활성화)
