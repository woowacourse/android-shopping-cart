## 데이터 모델 및 비즈니스 로직 개선
- [x] `Products` 내 `getPagedProducts` 범위 예외 처리

- [ ] 비동기 데이터 로딩 적용

- [x] `getProducts` 파라미터명 명확화
  - `ProductRepository`의 `limit` 변수가 '개수'인지 '끝 인덱스'인지 헷갈리지 않도록 직관적인 이름(`loadSize`)으로 변경
- [x] `hasNext` 내부 로직 개선
  - `InMemoryProductRepository`에서 `toList()`로 인한 불필요한 리스트 복사 발생 제거
