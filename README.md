# android-shopping-cart

- 사용자는 상품 목록의 상품을 장바구니에 추가할 수 있다.
    - 상품을 클릭하면 상품 상세로 이동한다.
    - 상품 상세에서 장바구니에 상품을 담을 수 있다.
    - 장바구니에서 원하는 상품을 삭제할 수 있다.

## 상품 목록 페이지

- [x] 상단 바에 장바구니 버튼이 있다.
    - [x] 장바구니 버튼을 클릭하면 장바구니 페이지로 이동한다.
- [x] 각 상품의 이미지와 상품이름, 상품가격이 있다.
- [x] 화면을 아래로 스크롤 할 수 있다.
- [] 상품 목록에서 장바구니에 담을 상품의 수를 선택 할 수 있다.
    - [] + 버튼을 누르면 장바구니에 상품이 추가되고 동시에 수량 선택 버튼이 노출된다
    - [] 상품 목록의 상품 수가 변화하면 장바구니에도 반영된다.

## 상품 상세 페이지

- [x] 클릭한 상품의 이미지, 상품 이름, 가격을 보여준다.
- [x] 장바구니 담기 버튼을 클릭하면 장바구니에 저장된다.
- [x] 상단 바의 x 버튼을 클릭하면 상품목록 페이지로 돌아온다.
- [] 장바구니에 담을 상품의 수를 선택할 수 있다.

## 장바구니 페이지

- [x] 장바구니에 담긴 상품들을 보여준다.
- [x] 상품을 삭제할 수 있는 버튼이 있다.
- [] 한 페이지에 5개의 상품이 보인다.
- [] 장바구니의 상품 수가 변화하면 상품 목록에도 반영된다.
